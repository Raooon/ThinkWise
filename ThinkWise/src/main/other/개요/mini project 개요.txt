Mini Project 2조

project명 : ThinkWise

project 개요 : 
학원 관리 시스템.
학원, 특히 미술학원의 수강생 등록, 강좌 등록 및 수강생의 수강 강좌 등 학원을 관리 하기 위한 학원 관리 시스템을 만들고자 함.

project 기능 구성
- 공통 사항 
 : 로그인, 로그아웃 - 로그인, 로그아웃 기능 구현 
 : 회원가입 페이지 - 회원 등록 기능 구현
 : 마이페이지 - 회원 정보 조회 및 수강생의 수강 강좌 조회

 : 회원(학생, 학부모) 관리 페이지 - 관리자 계정에서 회원 정보를 조회하고 관리할 수 있도록 기능 구현
 : 공지사항게시판 - 관리자 계정에서 학원의 공지사항을 출력하도록 구현
 : 학원 수업 스케줄 등록 페이지 - 학원의 수업 일정을 캘린더를 통해 조회할 수 있도록 구현
 : 월별 보고서 게시판 등록 페이지 - 수강생의 결과물을 출력하여 보여주도록 구현. 댓글 기능도 추가.
 : 학원 정보 페이지 - 학원의 정보와 학원의 위치를 지도를 통해 출력할 수 있도록 구현

project 팀명 : 예담보다낫조
project 팀장 : 이정민
